# 프로젝트 규칙

## 입력 제한
- 비디오 파일만 입력 가능 (.mp4, .mov, .avi, .mkv, .m4v)
- 1시간 이상 비디오는 처리할 수 없음 — 반드시 사전에 길이를 확인하고 초과 시 스킵할 것

## SRT 출력 규칙
- 생성된 SRT 파일은 반드시 맞춤법 검사를 수행해야 함
- 영어와 한국어가 혼용되는 구간은 병행 표기할 것
  - 예: `머신 러닝(Machine Learning)을 활용하여...`

## 개발 방식
- 앱 개발은 반드시 TDD(Test-Driven Development)로 진행할 것
  - 기능 구현 전에 테스트를 먼저 작성
  - 테스트 통과를 확인한 후 다음 단계로 진행
- 모든 변경사항은 반드시 검증 과정을 거쳐야 함
