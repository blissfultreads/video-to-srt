Video to SRT

폴더에 비디오를 넣고 클릭 한 번이면 자막 완성. OpenAI Whisper 기반 자동 SRT 생성기.

Python Whisper ffmpeg hunspell

GitHub Repository

주요 기능

🎬

폴더 일괄 처리

폴더 안의 모든 비디오 파일을 자동으로 탐색하고 하위 폴더까지 재귀 처리합니다.

🌏

한국어 + 영어

Whisper의 다국어 인식으로 한영 혼합 음성도 자동 감지하여 자막을 생성합니다.

✅

맞춤법 검사

hunspell 기반 맞춤법 자동 교정으로 더 정확한 자막을 만들어 줍니다.

🔤

영한 병행 표기

한국어 문장 속 영어 표현을 자동으로 감지하여 병행 표기합니다.

⏭️

중복 방지

이미 SRT 파일이 있는 비디오는 건너뛰어 재실행해도 안전합니다.

🖱️

원클릭 실행

.command 파일을 더블클릭하면 터미널 없이 바로 실행됩니다.

설치 방법

  1. 1
    필수 도구 설치 Python 3.10+, ffmpeg, Whisper, hunspell이 필요합니다.
  2. 2
    의존성 설치 아래 명령어를 터미널에 입력하세요.
# ffmpeg 설치 brew install ffmpeg # hunspell 및 한국어 사전 설치 brew install hunspell mkdir -p ~/Library/Spelling cd ~/Library/Spelling curl -LO https://cgit.freedesktop.org/libreoffice/dictionaries/plain/ko_KR/ko_KR.aff curl -LO https://cgit.freedesktop.org/libreoffice/dictionaries/plain/ko_KR/ko_KR.dic curl -LO https://cgit.freedesktop.org/libreoffice/dictionaries/plain/en/en_US.aff curl -LO https://cgit.freedesktop.org/libreoffice/dictionaries/plain/en/en_US.dic # Whisper 설치 pip install openai-whisper

사용 방법

# 폴더 지정 실행 python3 video_to_srt.py /path/to/videos # 현재 폴더 실행 python3 video_to_srt.py . # 원클릭 실행 (Finder에서 더블클릭) video_to_srt.command

옵션

옵션설명기본값
folder비디오 폴더 경로. (현재 폴더)
--modelWhisper 모델 (tiny/base/small/medium/large-v3)medium
--device디바이스 (cpu/cuda/mps)cpu
--no-spellcheck맞춤법 검사 건너뛰기검사 수행
